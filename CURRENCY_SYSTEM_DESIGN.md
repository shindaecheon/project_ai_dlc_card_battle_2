# 게임 화폐 시스템 설계: 골드 vs 다이아

## 📊 2중 화폐 시스템 개요

게임에는 **2가지 화폐**가 존재합니다:

| 화폐 | 타입 | 주요 용도 | 획득 방법 |
|------|------|----------|----------|
| **💰 골드 (Gold)** | 무료 화폐 | 배당, 아이템 구매 | 게임 플레이, 광고, 소량 현금 구매 |
| **💎 다이아 (Diamond)** | 프리미엄 화폐 | 프리미엄 아이템, 편의 기능 | 주로 현금 구매, 소량 무료 획득 |

---

## 💰 골드 (Gold) - 기본 화폐

### 역할
**게임의 핵심 경제 화폐** - 배틀 및 아이템 시스템의 중심

| **일일 무료 총합** | | | **~2,000G** |

| 패키지 | 골드량 | 가격 | 1G당 가격 |

| 항목 | 비용 | 설명 |
|------|------|------|
| **배당 베팅** | 50-500G | 핵심 소모처 |
| **아이템 구매** | 100-150G | 보호 방패, 회복 물약 등 |

### 특징
- ✅ 무료로 충분히 획득 가능
- ✅ 게임 플레이의 핵심 자원
- ✅ 소진 시 광고 시청으로 충전 가능
- ⚠️ 빠르게 소모됨 (배당 시스템)

## 🎯 골드 vs 다이아 비교

### 설계 철학

| 측면 | 골드 💰 | 다이아 💎 |
|------|---------|-----------|
| **획득 난이도** | 쉬움 (플레이 시 자동) | 어려움 (주로 과금) |
| **소모 속도** | 빠름 (배당마다 소진) | 느림 (선택적 사용) |
| **역할** | 필수 자원 (생존) | 선택 자원 (편의/강화) |
| **가치 인식** | 낮음 (소모품) | 높음 (프리미엄) |
| **과금 유도** | 간접적 (부족 시) | 직접적 (독점 콘텐츠) |

### 게임플레이 영향

```
무과금 유저:
- 골드: 풍부 (광고 시청 활용)
- 다이아: 매우 적음 (월 ~150개)
- 플레이: 기본 아이템만 사용
- 결과: 모든 콘텐츠 접근 가능하지만 시간 필요

저과금 유저 ($0.99-4.99/월):
- 골드: 여유 (구매 + 광고)
- 다이아: 적음 (~500개)
- 플레이: 가끔 프리미엄 아이템 사용
- 결과: 약간의 편의성 증가

중과금 유저 ($10-20/월):
- 골드: 충분 (구매)
- 다이아: 보통 (~1,500개)
- 플레이: 월간 패스 + 프리미엄 아이템
- 결과: 편안한 플레이 + 스킨 수집

고과금 유저 ($50+/월):
- 골드: 넉넉 (무제한)
- 다이아: 많음 (~5,000개)
- 플레이: 모든 프리미엄 기능 활용
- 결과: 최고 효율 + 완벽한 컬렉션
```

## 💡 다이아 시스템을 도입하는 이유

### 1. 수익 다각화

#### 골드만 있을 때의 문제
```
문제: 골드는 무료 획득이 쉬움
→ 과금 동기 약함
→ 광고 수익에만 의존
```

#### 다이아 도입 후
```
해결: 다이아 전용 콘텐츠 제공
→ 과금 동기 강화
→ IAP 수익 증가
→ 광고 + IAP 이중 수익
```

### 2. 가격 차별화

| 상품 | 가격 | 타겟 유저 |
|------|------|----------|
| 골드 1,000G | $0.99 | 가격 민감층 |
| 다이아 100개 | $0.99 | 프리미엄 추구층 |
| 월간 패스 | $4.99 | 장기 유저 |
| VIP 멤버십 | $19.99 | 고액 과금자 |

### 3. 심리적 가치 분리

```
골드:
"소모품" → "아깝지 않다" → "빨리 쓴다"
→ 빠른 소진 → 재구매 발생

다이아:
"귀한 자원" → "신중하게 쓴다" → "아껴둔다"
→ 느린 소진 → 축적 욕구 발생 → 더 구매
```

### 4. Pay to Win 방지

```
골드: 게임 승패에 직접 영향
→ 무과금도 충분히 획득 가능 (공정성 유지)

다이아: 편의성과 외형에만 영향
→ 과금해도 절대적 우위 없음 (밸런스 유지)
```

## 📊 다이아 경제 설계

### 다이아 가격 효율성 (의도적 설계)

| 구매 방식 | 효율성 | 설명 |
|----------|--------|------|
| **직접 다이아 구매** | ⭐⭐⭐⭐⭐ | 가장 효율적 (1다이아 = $0.0067) |
| **월간 패스 (다이아 결제)** | ⭐⭐⭐⭐☆ | 장기적으로 유리 |
| **다이아→골드 전환** | ⭐☆☆☆☆ | 매우 비효율 (권장 안 함) |

**의도**: 다이아를 직접 구매하도록 유도

### 다이아 소비 속도 조절

```javascript
// 무과금 유저: 월 150 다이아 획득
// 프리미엄 아이템 평균 가격: 70 다이아

월 150 다이아 ÷ 70 다이아/아이템 = 월 2회 사용

→ "부족하지만 쓸 수는 있다"는 느낌
→ "더 쓰고 싶으면 구매해야지" 유도
```

### 다이아 축적 방지

```
문제: 다이아를 너무 많이 모으면?
→ 더 이상 구매 안 함

해결책:
1. 한정 스킨 (기간 한정 판매)
2. 시즌 아이템 (매달 새로운 콘텐츠)
3. 다이아 소모 이벤트 (2배 보너스 등)
```

## 🎯 골드/다이아 밸런스 원칙

### 1. 공정성 (Fairness)

```
✅ 골드로 게임의 모든 핵심 기능 접근 가능
✅ 다이아는 "시간 단축" 또는 "외형"에만 사용
❌ 다이아 전용으로 강력한 카드 판매 금지
❌ 다이아 없으면 게임 진행 불가 금지
```

### 2. 명확한 역할 분리

| 화폐 | 주요 역할 | 금지 사항 |
|------|----------|----------|
| **골드** | 게임플레이 핵심 (배당, 아이템) | 외형 커스터마이징 구매 |
| **다이아** | 편의성, 외형, 프리미엄 | 게임 승패에 직접 영향 |

### 3. 전환 비효율성 유지

```
다이아 → 골드: 비효율 (1:10)
→ 다이아를 골드 대신 쓰지 않도록

골드 → 다이아: 불가능
→ 다이아의 프리미엄 가치 유지
```
