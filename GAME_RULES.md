# 카드 배틀 게임 규칙 문서

## 📋 게임 개요

**장르**: 1:1 턴제 카드 배틀 게임  
**플랫폼**: 웹 브라우저 (HTML5)  
**플레이어**: 플레이어 vs AI

---

## 🎮 핵심 게임 메커니즘

### 1. 게임 초기화
- **덱 구성**: 표준 52장 트럼프 카드 (스페이드♠, 하트♥, 다이아♦, 클로버♣ 각 13장)
- **카드 배분**: 
  - 플레이어: 10장
  - AI: 10장
  - 셔플 후 무작위 배분
- **초기 HP**: 플레이어 10, AI 10
- **턴 제한**: 최대 10턴

### 2. 카드 시스템

#### 카드 구성
| 랭크 | 값 |
|------|-----|
| A    | 1   |
| 2-10 | 숫자 그대로 |
| J    | 11  |
| Q    | 12  |
| K    | 13  |

#### 카드 속성
- **무늬**: ♠ (스페이드), ♥ (하트), ♦ (다이아), ♣ (클로버)
- **색상**: 빨강(♥, ♦), 검정(♠, ♣)
- **현재**: 무늬는 시각적 표현만, 게임 로직에 영향 없음

### 3. 게임 플레이 흐름

```
게임 시작
  ↓
[턴 시작]
  ↓
플레이어 카드 선택
  ↓
AI 카드 자동 선택
  ↓
배틀 실행 (숫자 비교)
  ↓
결과 판정 (HP 감소)
  ↓
승패 확인
  ↓
[다음 턴] 또는 [게임 종료]
```

### 4. 배틀 시스템

#### 승패 판정
- **플레이어 승리**: 플레이어 카드 값 > AI 카드 값 → AI HP -1
- **플레이어 패배**: 플레이어 카드 값 < AI 카드 값 → 플레이어 HP -1
- **무승부**: 플레이어 카드 값 = AI 카드 값 → HP 변화 없음

#### 게임 종료 조건
1. 플레이어 HP가 0 이하
2. AI HP가 0 이하
3. 양측 카드가 모두 소진 (10턴 완료)

#### 최종 승리 조건
- AI HP가 0 → 플레이어 승리
- 플레이어 HP가 0 → 플레이어 패배
- 10턴 후: HP가 더 높은 쪽 승리
- HP 동일 → 무승부

### 5. AI 전략

현재 AI는 **최소 손실 전략** 사용:

1. **이길 수 있는 경우**: 플레이어보다 큰 카드 중 **가장 낮은 값** 사용
   - 예: 플레이어가 7을 냈다면 → AI는 8, 9, 10 중 8을 선택
2. **이길 수 없는 경우**: **가장 낮은 카드를 버림**
   - 예: 플레이어가 K를 냈다면 → AI는 A 또는 2 같은 최저값 버림

## 🎯 게임 밸런스 분석

### 현재 밸런스 특성
- **운 요소**: 70% (초기 카드 배분이 승패에 큰 영향)
- **전략 요소**: 30% (카드 순서 선택, 상대 카드 추론)

### 밸런스 이슈
1. **AI가 플레이어 카드를 본 후 선택** → AI에게 유리
2. **무늬/속성 시스템 없음** → 전략 깊이 부족
3. **카드 효과 없음** → 단순 숫자 비교만
4. **10장 고정** → 게임 길이 예측 가능, 긴장감 부족

## 💡 확장 가능 규칙 (향후 적용)

### Phase 1: 속성 시스템
- 무늬 상성 추가 (예: ♠ > ♥ > ♦ > ♣ > ♠)
- 같은 무늬 연속 사용 시 보너스

### Phase 2: 특수 카드
- **A**: 1 또는 14로 사용 가능 (플레이어 선택)
- **J**: 상대 다음 카드 1장 공개
- **Q**: 다음 턴 데미지 2배
- **K**: 라운드 승리 시 HP +1 회복

### Phase 3: 스킬 시스템
- 매 3턴마다 특수 스킬 사용 가능
- 예: 카드 교환, HP 회복, 상대 카드 제한

### Phase 4: 덱 빌딩
- 게임 시작 전 30장 중 10장 선택
- 레어 카드 추가 (골드 카드, 강화 카드)

## 🔧 기술 스택

- **프론트엔드**: HTML5, CSS3, Vanilla JavaScript
- **렌더링**: DOM 직접 조작
- **상태 관리**: 전역 변수
- **애니메이션**: CSS Transition

## 📈 개선 방향

1. **게임 규칙 다양화**: 속성 시스템, 특수 카드 효과
2. **AI 난이도 단계**: Easy (랜덤), Normal (현재), Hard (카드 카운팅)
3. **비주얼 개선**: 카드 애니메이션, 전투 이펙트
4. **사운드**: 카드 플레이, 승패 효과음
5. **통계**: 승률, 평균 HP, 최고 연승 기록
