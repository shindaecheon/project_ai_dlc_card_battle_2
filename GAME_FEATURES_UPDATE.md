# 게임 기능 업데이트 (2026-05-13)

## 🎉 주요 신규 기능

### 1. 모드 선택 시스템 ⭐ 신규

게임 시작 시 3가지 모드 중 선택:

#### A. VS AI 모드 (기본)
- AI와 대전
- AI 감정 시스템 적용
- 전략적 AI 플레이
- 단일 플레이어 경험

#### B. 유저 VS 유저 (PvP)
- **같은 화면 2인 대전**
- 턴 전환 시 화면 가림 처리
- 공정한 1:1 대결
- 친구와 로컬 대전 가능

**수익화 연계**:
- 온라인 매칭 추가 시 매칭 비용 부과 가능
- 프렌드 배틀 토큰 판매
- 스킨/이모티콘 표현 욕구 증가

#### C. 야생 모드 (Wild Mode) ⭐ 완전 신규

**컨셉**: 매 턴 예측 불가능한 랜덤 이벤트 발동

| 이벤트 | 아이콘 | 효과 | 전략적 영향 |
|--------|--------|------|------------|
| **역전의 룰** | 🔄 | 이번 턴 숫자 낮은 쪽 승리 | 약한 카드가 강해짐 |
| **점쟁이** | 🔮 | AI 카드 미리 공개 | 완벽한 정보 플레이 |
| **독안개** | ☠️ | 패배 시 골드 2배 손실 | 극도의 신중함 필요 |
| **세금 징수** | 💸 | 양측 100G 강제 차감 | 배당 전략 무효화 |
| **황금비** | ✨ | 승자에게 +150G 보너스 | 공격적 플레이 유도 |

**밸런스**:
- 5종 이벤트 랜덤 발동
- 긍정/부정 이벤트 균형
- 전략과 운의 절묘한 조화

**수익화 연계**:
- 이벤트 스킵권 (골드 200G)
- 유리한 이벤트 선택권 (다이아 전용, 향후)
- 이벤트 히스토리 통계 (프리미엄 기능)

### 2. AI 감정 시스템 ⭐ 신규

**컨셉**: AI의 골드 상태에 따른 실시간 감정 표현

#### 감정 7종

| 감정 | 조건 | 아이콘 | 애니메이션 | 심리적 효과 |
|------|------|--------|------------|------------|
| **평온** | 골드 균형 (±200G) | 😐 | 없음 | 중립적 긴장감 |
| **기분좋음** | AI +200~400G 우세 | 😊 | 플로팅 | 플레이어 압박감 |
| **자신만만** | AI +400~600G 우세 | 😎 | 펄스 | 플레이어 위기감 |
| **여유** | AI +600G 이상 | 😁 | 플로팅 강화 | 플레이어 절망감 |
| **짜증** | AI -200~400G 열세 | 😠 | 떨림 | 플레이어 희망 |
| **분노** | AI -400~600G 열세 | 😡 | 떨림 강화 | 플레이어 승기 |
| **초조** | AI 300G 이하 | 😰 | 떨림 강화 | 플레이어 승리 확신 |

**기술 구현**:
```css
/* 떨림 애니메이션 */

    0%, 100% { transform: translateY(0); }
```

**심리적 가치**:
- 플레이어에게 AI 상태 피드백
- 감정 이입 유도
- 승패 긴장감 증폭
- 역전 드라마 강화

**수익화 연계**:
- AI 감정 커스터마이징 (스킨)
- 감정 표현 확장팩 (다양한 이모티콘)
- AI 성격 선택 (공격적/보수적/도발적)

### 3. 올인 아이템 ⭐ 신규

**아이템 정보**:
- 아이콘: 💀
- 이름: 올인 (All-in)
- 효과: 현재 소지 골드 전액 베팅
- 비용: 무료
- 제한: 게임당 1회만 사용 가능
- 조건: 골드 500G 이하일 때만 사용 가능

**메커니즘**:
```javascript
// 올인 사용 시
if (playerGold <= 500 && !usedItems.includes('allin')) {
    currentBet = playerGold;  // 전액 베팅
    // 승리: playerGold × 2
    // 패배: 즉시 파산
}
```

**전략적 가치**:
- 파산 직전 마지막 역전 카드
- 극적인 반전 가능
- 고위험/고수익 선택
- 심리적 긴장감 극대화

**사용 시나리오**:
1. 골드 300G, AI 800G → 올인으로 600G 역전 가능
2. 골드 150G, 절체절명 → 올인 승리 시 300G 회복
3. 강력한 특수 카드 보유 시 올인으로 승부 결정

**수익화 연계**:
- 올인 재사용권 (게임당 1회 → 2회)
- 올인 보험 (패배 시 골드 50% 보존)
- 올인 성공률 향상 아이템

### 4. 특수카드 고정 배분 ⭐ 신규

**변경 사항**:
- **이전**: 52장 덱에서 완전 랜덤 10장 배분
- **현재**: J/Q/K 3장 + 일반카드 7장 고정

**배분 로직**:
```javascript
function dealHand() {
    const deck = createDeck();
    const specials = shuffle(deck.filter(c => ['J', 'Q', 'K'].includes(c.rank)));
    const normals = shuffle(deck.filter(c => !['J', 'Q', 'K'].includes(c.rank)));
    
    const playerSpecials = specials.slice(0, 3);  // 특수 3장
    const aiSpecials = specials.slice(3, 6);
    const playerNormals = normals.slice(0, 7);    // 일반 7장
    const aiNormals = normals.slice(7, 14);
    
    return {
        player: shuffle([...playerSpecials, ...playerNormals]),
        ai: shuffle([...aiSpecials, ...aiNormals])
    };
}
```

**효과**:
- 특수 능력 보장으로 전략성 증가
- 게임마다 일관된 카드 구성
- 운 요소 감소, 실력 비중 증가
- 신규 유저도 특수 능력 체험 보장

**밸런스 개선**:
- 특수 카드 0장 악운 방지
- 특수 카드 과다 보유 방지
- 공정한 대전 환경

### 5. ♠J 능력 변경

**변경 사항**:
- **이전**: 엿보기 (AI 카드 미리 확인)
- **현재**: 압박 (AI 최약 카드 강제 출전)

**변경 이유**:
- 엿보기: 수동적 정보 획득
- 압박: 능동적 승리 확정
- 더 직관적이고 강력한 효과

**효과 비교**:
| 능력 | 장점 | 단점 |
|------|------|------|
| 엿보기 (이전) | 정보 우위, 최적 카드 선택 | 여전히 이길 카드 필요 |
| 압박 (현재) | 거의 승리 보장 | 운 요소 감소 우려 |

**밸런스 조정**:
- ♠J는 여전히 A에게 패배
- AI도 ♠J 보유 가능 (공정)
- 1게임 최대 1회 사용

---

### 6. 아이템 1턴 1회 제한 ⭐ 신규

**변경 사항**:
- **이전**: 턴당 다수 아이템 동시 사용 가능
- **현재**: 1턴에 1개만 사용 가능

**효과**:
- 아이템 조합 불가 (독 바늘 + 도박꾼의 주화 동시 사용 차단)
- 전략적 우선순위 선택 필요
- 아이템 가치 재조정

**구현**:
```javascript
let activeItem = null;  // 현재 턴 사용 아이템

function useItem(item) {
    if (activeItem !== null) {
        showMessage("⚠️ 이번 턴에 이미 아이템을 사용했습니다!");
        return;
    }
    activeItem = item.id;
    // 아이템 효과 적용
}
```

**밸런스 개선**:
- 과도한 아이템 조합 방지
- 게임 길이 적정 유지
- 아이템 가격 재조정 불필요

## 🎮 게임 플레이 변화

### 전략 다양화

**이전 (단순)**:
1. 배당 선택
2. 아이템 다수 사용
3. 카드 내기
4. 결과 확인

**현재 (복잡)**:
1. **모드 선택** (VS AI / PvP / 야생)
2. 배당 선택
3. **1개 아이템 전략적 선택** (1턴 1회)
4. 카드 내기 (특수카드 3장 보장)
5. **야생 이벤트 대응**
6. 결과 확인
7. **AI 감정 관찰**
8. **올인 타이밍 판단**

### 리플레이 가치 증가

- **모드별 경험 차별화**: VS AI vs PvP vs 야생
- **AI 감정 관찰의 재미**: 매 게임 다른 감정 흐름
- **야생 이벤트 랜덤성**: 예측 불가능한 전개
- **올인 극적 반전**: 절체절명 순간의 카타르시스
