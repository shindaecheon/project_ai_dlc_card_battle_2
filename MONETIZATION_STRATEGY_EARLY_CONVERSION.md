# 초기 과금 전환 전략 (5~30분 내 PU 전환)

## 🎯 목표

**게임 플레이 5~30분 내에 무과금 유저를 과금 유저(PU)로 전환**  
전환율 목표: 25~35% (업계 평균 15~20%보다 높은 목표)

---

## 📊 현재 게임 구조 분석

### 플레이 타임라인

| 시간 | 활동 | 유저 상태 |
|------|------|----------|
| **0-2분** | 튜토리얼 + 첫 판 | 학습 중, 흥미 형성 |
| **2-5분** | 2-3판 완료 | 게임 이해, 재미 체감 |
| **5-10분** | 4-6판 완료 | 아이템 소진 시작 |
| **10-15분** | 7-10판 완료 | 골드/아이템 부족 체감 |
| **15-30분** | 11-20판 완료 | 전략 확립, 한계 느낌 |

### 현재 자원 상태 (10분 후)

| 자원 | 초기 | 5분 후 (3판) | 10분 후 (6판) | 15분 후 (10판) |
|------|------|-------------|--------------|---------------|
| **골드** | 1,000G | ~800G | ~500G | ~200G |
| **아이템** | 12개 | ~8개 | ~4개 | ~0개 |
| **턴 경험** | 0턴 | ~21턴 | ~42턴 | ~70턴 |

**결론**: 10-15분 시점에 자원 부족 위기 → **과금 전환 최적 타이밍**

**전략**:
1. ✅ **초기 아이템 체험** (이미 구현됨)
   - 각 2개씩 지급 → 사용 유도
   - 효과 체감 → 가치 인지

2. 🆕 **첫 승리 보너스**
   ```
   첫 판 승리 시:
   - 보너스 골드 +300G
   ```

## 🧠 심리학적 전환 트리거

### 1. 손실 회피 (Loss Aversion)

```
현재 상황 강조:
"지금 파산하면 지금까지의 노력이 물거품!"
"100G만 있으면 역전 가능합니다"

→ 긴급 충전 제안 ($0.99)
```

### 2. 앵커링 (Anchoring Effect)

```
정상가: $4.99 (취소선)
특가: $0.99 (80% 할인! - 빨간색 강조)

→ "$0.99는 저렴하다"는 인식 형성
```

### 3. 희소성 (Scarcity)

```
⏰ 남은 시간: 4분 57초
🔥 한정 수량: 97/100 판매
⭐ 1회 한정 제안

→ "지금 아니면 못 산다" 긴급성 조성
```

### 4. 상호성 (Reciprocity)

```
무료 지급:
- 초기 아이템 12개
- 첫 승리 보너스 300G
- 광고 시청 시 500G

→ "이렇게 많이 줬는데, 조금은 지불해야지" 심리
```

### 5. 사회적 증거 (Social Proof)

```
"87%의 플레이어가 선택!"
"인기 상품 #1"
"친구 3명이 이미 구매했습니다"

→ "다들 사니까 나도" 심리
```

### 6. 진행 효과 (Progress Effect)

```
업적 진행도:
[▓▓▓▓▓▓▓░░░] 70%

"70%까지 왔는데 포기하시겠어요?"

→ 완성욕 자극
```

- **트리거**: 월간 패스 30일 사용 후
- **제안**: "VIP 회원으로 업그레이드하시겠습니까?"

  - 일일 골드 2배

- 인위적 어려움 조작 (Pay to Win 강요)
