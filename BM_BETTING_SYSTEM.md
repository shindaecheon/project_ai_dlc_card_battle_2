# 배당 시스템 기반 비즈니스 모델 (BM)

## 🎯 핵심 컨셉

**"리스크 vs 리워드"** - 배당 시스템을 중심으로 한 경제 게임플레이  
현재 구현된 배당 시스템(50G/100G/200G)을 확장하여 지속 가능한 수익 모델 구축

## 💰 배당 시스템 경제 설계

### 1. 현재 시스템 분석

| 요소 | 현재 상태 | 개선 방향 |
|------|----------|----------|
| 초기 골드 | 1,000G | 적정 (20턴 분량) |
| 배당 옵션 | 50/100/200G | 확장 필요 (5단계로) |
| 배당 비율 | 1:1 (100% 환수) | 다양화 필요 |
| 위험 관리 | 없음 | 파산 방지 시스템 추가 |
| 경제 순환 | 단순 | 다층 경제 구조 |

### 2. 현재 배당 시스템 (구현 완료)

#### A. 기본 배당 구조 (3단계) ✅ 구현됨

| 레벨 | 배당액 | 배당률 | 위험도 | 권장 상황 |
|------|--------|--------|--------|----------|
| **안전** | 50G | 2.0배 | ⭐⭐☆☆☆ | 안정적 플레이 |
| **표준** | 100G | 2.0배 | ⭐⭐⭐☆☆ | 일반적 선택 |
| **공격** | 200G | 2.0배 | ⭐⭐⭐⭐☆ | 승리 확신 시 |

**배당률 설명**:
- 승리 시: 배당액 × 배당률 획득 (예: 100G 베팅 시 200G 획득)
- 패배 시: 배당액 전액 손실
- 무승부: 배당액 회수

#### B. 올인 아이템 ✅ 구현됨

##### 💀 올인 (All-in)
- **현재 소지 골드 전액 베팅**
- 승리 시: 베팅액 × 2배 획득
- 패배 시: 즉시 파산
- **게임당 1회만 사용 가능**
- **사용 조건**: 골드 500G 이하일 때만
- 비용: 무료 (아이템 슬롯 제공)

**전략적 가치**:
- 파산 직전 역전 카드
- 고위험/고수익 극단적 선택
- 심리적 긴장감 극대화

#### C. 특수 배당 모드 (향후 확장안)

##### 🎲 리스크 베팅 (Risk Bet) - 미구현
- 배당액의 **150%를 걸고** 승리 시 **300% 보상**
- 예: 100G 리스크 베팅 → 150G 손실 위험 / 300G 획득 가능
- 1일 3회 제한 (무료), 추가 사용은 골드 500G

##### 💎 프리미엄 베팅 (Premium Bet) - 미구현
- **골드 100G 소모** → 패배해도 골드 손실 50% 감소
- 승리 시 보너스 골드 +20%
- 월간 패스 구매 시 무제한 사용

##### 🎰 잭팟 베팅 (Jackpot Bet) - 미구현
- 배당액의 10%를 잭팟 풀에 자동 적립
- 5연승 시 잭팟 풀 전액 획득
- 전체 유저 공유 풀로 최대 100,000G까지 누적

## 🎲 배당 시스템을 활용한 게임 모드 ✅ 구현됨

### 1. VS AI 모드 (기본) ✅ 구현됨
- 초기 골드: 1,000G
- 배당 선택: 50G / 100G / 200G
- AI 감정 시스템 적용
- 골드 소진 시 일일 무료 충전 500G 제공 (향후)
- **수익 모델**: 광고 (골드 충전 시), IAP (골드 구매)

### 1-1. 유저 VS 유저 모드 (PvP) ✅ 신규 구현
- 같은 화면에서 2인 대전
- 턴 전환 시 화면 가림 처리
- 공정한 1:1 대결
- AI 감정 시스템 비활성화
- **수익 모델**: 토너먼트 입장료 (향후), 스킨 판매

### 1-2. 야생 모드 (Wild Mode) ✅ 신규 구현
- 매 턴 랜덤 이벤트 발동 (5종):
  - 🔄 **역전의 룰**: 숫자 낮은 쪽이 승리
  - 🔮 **점쟁이**: AI 카드 미리 공개
  - ☠️ **독안개**: 이번 턴 패배 시 골드 2배 손실
  - 💸 **세금 징수**: 양측 100G 강제 차감
  - ✨ **황금비**: 승자에게 추가 150G 보너스
- 고위험/고수익 게임플레이
- 예측 불가능한 재미
- **수익 모델**: 이벤트 스킵권 판매 (향후), 유리한 이벤트 선택권

### 2. 랭크전 모드 - 향후 구현
- 입장료: 500G
- 승리 시 750G + 랭크 포인트 +10
- 패배 시 골드 손실 없음 (입장료만)
- 티어별 보상 (브론즈 → 실버 → 골드 → 플래티넘 → 다이아)
- **수익 모델**: 입장료 반복 소모

### 3. 토너먼트 모드 - 향후 구현
- 입장료: 1,000G
- 8인 토너먼트, 단일 패배 탈락
- 상금 풀: 입장료의 80% (플랫폼 수수료 20%)
- 1위: 4,000G, 2위: 2,000G, 3-4위: 1,000G
- **수익 모델**: 플랫폼 수수료 + 재참가 유도

### 4. 하이롤러 모드 (고배당 전용) - 향후 구현
- 최소 배당: 500G
- 최대 배당: 5,000G
- 고수 유저 대상, 높은 리스크/리워드
- VIP 전용 (일정 랭크 이상 or 월간 패스 보유자)
- **수익 모델**: 진입 장벽으로 과금 유도

### 5. 연습 모드 (무료) - 향후 구현
- 골드 없이 플레이 가능
- AI 난이도 선택 (Easy / Normal / Hard)
- 보상 없음, 순수 연습용
- **수익 모델**: 신규 유저 온보딩 후 본 게임 전환

## 🎯 배당 시스템 최적화 전략

### 1. 심리적 설계 (Loss Aversion 활용)

#### A. 연패 방지 시스템
- **3연패 시**: 다음 판 배당금 50% 할인 쿠폰 제공
- **5연패 시**: 무료 500G 지급 + "역전의 기회" 이벤트
- **10연패 시**: 보장 승리 AI (Easy 난이도 자동 매칭)

**목적**: 이탈 방지, 과금 전 게임 재미 유지

#### B. 승리 연속 보너스
- **3연승**: 다음 판 배당금 +20%
- **5연승**: 다음 판 배당금 +50%
- **10연승**: 레전더리 카드팩 1개 무료

**목적**: 게임 몰입도 증가, 긍정적 경험 강화

### 2. 배당 선택 가이드 (AI 어시스턴스)

#### 승률 예측 시스템
- 플레이어의 패 vs AI 예상 패 분석
- 승률 % 표시 (예: "이 패로 이길 확률 65%")
- 추천 배당액 제시 (예: "안전하게 50G 추천")

**수익화 연계**:
- 무료 유저: 모호한 예측 (50~70%)
- 월간 패스 유저: 정확한 예측 (± 5% 오차)
- 골드 100G: 1회 정확한 AI 카드 공개

### 3. 소셜 배당 시스템

#### A. 친구 대결
- 친구와 골드를 걸고 대결
- 수수료 10% (플랫폼 수익)
- 예: 친구와 500G 베팅 → 승자 900G, 패자 -500G, 플랫폼 +100G

#### B. 관전 베팅
- 타인의 경기를 보며 결과에 베팅
- 배당률 2.0배 (예측 맞으면 2배 획득)
- 1일 5회 무료, 추가 베팅은 골드 200G

| **일일 손실 한도** | 1일 최대 5,000G 손실 시 배당 중지, 다음 날 재개 |
| **연속 플레이 경고** | 2시간 연속 플레이 시 "휴식 권장" 팝업 |
| **청소년 보호** | 만 18세 미만 최대 배당 50G 제한 |
| **자가 제한** | 유저가 스스로 일일/주간 배당 한도 설정 가능 |

### 2. 공정성 보장

- **AI 카드 선택 투명화**: 게임 종료 후 AI 전략 로그 공개
- **난이도 알고리즘 공개**: AI 승률 50%로 조정 (완전 랜덤 아님)
- **확률 공개**: 카드팩 드롭률 법적 의무 준수
- **환불 정책**: 버그로 인한 골드 손실 시 100% 보상

### 3. Pay to Win 방지

- 골드는 **시간 단축**이지 절대 우위 아님
- 모든 카드는 무료로 획득 가능 (시간 투자 필요)
- 랭크전은 "덱 가치 제한 모드" 제공 (커먼 카드만 사용)
- 토너먼트는 동일 조건 (모두 1,000G 시작)

### 우리 게임 vs 경쟁사

| 요소 | 우리 게임 | 하스스톤 | Marvel Snap |
|------|----------|----------|-------------|
| **핵심 재화** | 골드 (배당 중심) | 골드/더스트 (카드팩) | 크레딧/골드 |
| **경제 순환** | 빠름 (턴마다 소모) | 느림 (팩 구매 중심) | 중간 (배틀패스) |
| **리스크 요소** | 높음 (배당 손실) | 낮음 (카드만 뽑기) | 중간 (큐브) |
| **전략 깊이** | 배당 타이밍 | 덱 빌딩 | 리트리트 판단 |
| **진입 장벽** | 낮음 (3분 학습) | 높음 (복잡한 룰) | 중간 |
| **과금 압박** | 중간 (광고 대체) | 높음 (카드 필수) | 낮음 (스킨 중심) |

**우리의 강점**: 
- ⚡ 빠른 경제 순환 → 높은 몰입도
- 🎲 명확한 리스크/리워드 → 전략적 재미
- 💰 배당 시스템 → 도박성 아닌 전략적 선택

## 🎯 결론 및 실행 계획

### 배당 시스템이 BM의 핵심인 이유

1. **높은 몰입도**: 턴마다 배당 선택 → 지속적 의사결정
2. **자연스러운 소모**: 승패와 관계없이 골드 순환
3. **심리적 몰입**: "다음 판에 역전" 심리 활용
4. **다양한 수익 모델**: 골드, 다이아, 패스, 토너먼트 등
5. **확장성**: 새로운 배당 모드 무한 추가 가능

### 즉시 실행 가능한 액션 아이템

**단기 (1개월 내)**:
1. 5단계 배당 시스템 구현 (10/50/100/200/500G)
2. 배당률 차등화 (1.5배 ~ 2.5배)
3. 광고 연동 (골드 충전 500G)
4. 기본 IAP 설정 (골드 패키지 5종)
5. 파산 방지 시스템 (일일 무료 500G)

**중기 (3개월 내)**:
1. 특수 배당 모드 3종 (리스크/프리미엄/잭팟)
2. 랭크전 + 토너먼트 시스템
3. 월간 패스 출시
4. 승률 예측 AI 어시스턴스
5. 소셜 기능 (친구 대결)
