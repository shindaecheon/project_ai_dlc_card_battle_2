# 카드 배틀 게임 규칙

## 기본 개요
- 카드 대전 게임
- 각 플레이어는 10장의 카드를 보유하고 시작 (특수카드 3장 + 일반카드 7장 고정)
- 총 10턴 동안 진행
- 승패는 골드로 결정 (HP 없음)

---

## 게임 모드

| 모드 | 설명 |
|------|------|
| 👥 유저 VS 유저 | 같은 화면에서 2인 대전 (턴 교대 시 화면 가림) |
| 🤖 VS AI | AI와 1:1 대전 |
| 🌪️ 야생 | AI 대전 + 매 턴 랜덤 이벤트 발동 |

---

## 카드 구성

### 덱 구성
- 표준 52장 트럼프 카드 (조커 제외)
- 문양: ♠(스페이드), ♥(하트), ♦(다이아), ♣(클로버)

### 카드 배분 규칙
- 특수카드(J/Q/K) 12장 중 랜덤 3장씩 고정 배분
- 일반카드(A~10) 중 랜덤 7장씩 배분
- 나머지는 잔여 덱으로 보관

### 카드 숫자 값
| 카드 | 값 |
|------|-----|
| A | 1 |
| 2~10 | 숫자 그대로 |
| J | 11 |
| Q | 12 |
| K | 13 |

---

## 게임 진행

### 시작 조건
- 양측 소지금: 1,000 골드로 시작

### 턴 진행
1. 플레이어가 배당액을 선택 (50 / 100 / 200 골드)
2. (야생 모드) 랜덤 이벤트 발동
3. 아이템 사용 가능 (턴당 1회)
4. 플레이어가 자신의 패에서 카드 1장을 선택하여 제출
5. AI/상대가 카드 1장을 제출
6. 양측 카드의 숫자 값을 비교
7. 결과에 따라 배당금 정산
8. 다음 턴으로 진행

### 배당 시스템
| 배당액 | 설명 |
|--------|------|
| 50 골드 | 안전한 베팅 |
| 100 골드 | 중간 베팅 |
| 200 골드 | 공격적 베팅 |

#### 배당 규칙
- 카드를 내기 전에 배당액을 먼저 선택
- 승리한 쪽이 배당액만큼 획득, 패배한 쪽은 배당액만큼 손실
- 무승부 시 변동 없음
- 소지금이 배당액보다 적으면 해당 배당액 선택 불가

## 승패 판정 (턴별)

### 기본 규칙
| 조건 | 결과 |
|------|------|
| 플레이어 카드 > AI 카드 | 플레이어 승리 (배당금 획득) |
| 플레이어 카드 < AI 카드 | AI 승리 (플레이어 배당금 손실) |
| 플레이어 카드 = AI 카드 | 문양 우선순위로 판정 |

### 특수 규칙: A(에이스)의 역전
- A는 K, Q, J를 상대할 경우 **무조건 승리**
- A vs A인 경우 문양 우선순위로 판정
- A vs 2~10인 경우 기본 숫자 비교 (A=1로 계산, 패배)

### 문양 우선순위 (숫자가 같을 경우 적용)
| 순위 | 문양 | 기호 |
|------|------|------|
| 1위 (최강) | 스페이드 | ♠ |
| 2위 | 다이아 | ♦ |
| 3위 | 하트 | ♥ |
| 4위 (최약) | 클로버 | ♣ |

> 예시: ♠7 vs ♣7 → 스페이드 승리 (♠ > ♣)

---

## 게임 종료 조건

다음 중 하나라도 해당되면 게임 종료:
1. 한쪽 골드가 0 이하 (파산)
2. 10턴 모두 소진 (카드가 모두 사용됨)

### 최종 승패 판정
| 조건 | 결과 |
|------|------|
| AI 골드 ≤ 0 | 플레이어 승리 (AI 파산) |
| 플레이어 골드 ≤ 0 | 플레이어 패배 (파산) |
| 10턴 종료 후 플레이어 골드 > AI 골드 | 플레이어 승리 |
| 10턴 종료 후 플레이어 골드 < AI 골드 | 플레이어 패배 |
| 10턴 종료 후 골드 동일 | 무승부 |

## J / Q / K 특수 능력

J, Q, K 카드는 문양에 따라 고유 능력을 보유합니다.
능력은 해당 카드를 제출할 때 자동으로 발동됩니다.
각 플레이어는 게임 시작 시 특수카드 3장을 고정으로 받습니다.

### ♠ 스페이드 — "전투/공격" 테마

| 카드 | 능력명 | 효과 |
|------|--------|------|
| ♠J | 선제공격 | 이번 턴 내 카드 숫자 +3 |
| ♠Q | 증폭 | 승리 시 배당금 2배 획득 |
| ♠K | 압도 | 숫자 비교 무시, 무조건 승리 (A에게는 패배) |

### ♦ 다이아 — "재물/경제" 테마

| 카드 | 능력명 | 효과 |
|------|--------|------|
| ♦J | 도둑 | 승리 시 배당금 1.5배 획득 |
| ♦Q | 절약 | 배당금을 걸지 않고 무료 승부 가능 |
| ♦K | 위엄 | 이번 턴 배당액 강제 2배 (양측 모두 적용) |

### ♥ 하트 — "생명/수호" 테마

| 카드 | 능력명 | 효과 |
|------|--------|------|
| ♥J | 회피 | 패배해도 골드 손실 없음 |
| ♥Q | 치유 | 승패와 관계없이 +100G 회복 |
| ♥K | 처형 | 상대 골드 300 이하일 때 사용하면 즉시 파산 승리 |

### ♣ 클로버 — "혼돈/트릭" 테마

| 카드 | 능력명 | 효과 |
|------|--------|------|
| ♣J | 반격 | 패배 시 손실 골드의 50%를 회수 |
| ♣Q | 혼란 | 이번 턴 문양 우선순위 역전 (♣>♥>♦>♠) |
| ♣K | 봉쇄 | 상대 다음 턴 특수 능력(J/Q/K) 사용 불가 |

### 밸런스 설계

| 문양 | 강점 | 약점 |
|------|------|------|
| ♠ | 순수 전투력 최강 | 경제/회복 수단 없음 |
| ♦ | 골드 수급 유리 | 직접적 전투력 낮음 |
| ♥ | 생존력 최고 | 공격력 부족 (처형은 조건부) |
| ♣ | 피해 경감/규칙 변형 | 자체 공격력 낮음 |

> 참고: A는 J/Q/K를 만나면 무조건 승리하므로, 특수 능력 카드의 천적 역할

## 아이템 시스템

우측 하단에 아이템 슬롯이 위치하며, 턴당 1개만 사용 가능합니다.
각 아이템은 게임당 1회 사용 가능합니다.

| 아이템 | 아이콘 | 효과 | 비용 |
|--------|--------|------|------|
| 보호 방패 | 🛡️ | 이번 턴 패배해도 골드 손실 없음 | 150G |
| 회복 물약 | 🧪 | 즉시 200G 회복 | 100G |
| 독 바늘 | 🗡️ | 이번 턴 승리 시 배당금 2배 획득 | 150G |
| 수정 구슬 | 🔮 | AI 패 카드 1장 확인 | 100G |
| 교체 카드 | 🔄 | 손패 1장을 버리고 덱에서 새로 1장 드로우 | 150G |
| 도박꾼의 주화 | 🎲 | 이번 턴 배당금 3배 (승패 모두 적용) | 100G |
| 올인 | 💀 | 소지금 전액 베팅 (골드 500 이하일 때만 사용 가능) | 무료 |

## AI 전략

### 기본 모드 (블라인드)
- AI는 플레이어 카드를 모른 채 중간값 위주로 랜덤 선택

### 위기 모드 (골드 100 이하)
- AI가 몰래 플레이어 카드를 확인하고 최적 대응
- 이길 수 있으면 → 최소 비용으로 승리
- 이길 수 없으면 → 가장 낮은 카드를 버림

### AI 감정 시스템
AI의 상태에 따라 표정이 변합니다:

| 감정 | 조건 | 표정 | 연출 |
|------|------|------|------|
| 평온 | 기본 상태 | 😐 | 없음 |
| 기분 좋음 | 2연승 | 🙂 | 없음 |
| 자신만만 | 3연승 이상 | 😎 | 둥둥 떠다님 |
| 여유 | 골드 2배 이상 앞설 때 | 😏 | 둥둥 떠다님 |
| 짜증남 | 2연패 | 😤 | 붉은 펄스 |
| 분노 | 3연패 이상 | 😡 | 붉은 펄스 |
| 초조 | 골드 300 이하 | 😰 | 부들부들 떨림 |

## 야생 모드 — 라운드 이벤트

야생 모드에서는 매 턴 배당 선택 후 5종의 이벤트 중 하나가 랜덤 발동됩니다.

| 이벤트 | 아이콘 | 효과 |
|--------|--------|------|
| 역전의 룰 | 🔄 | 이번 턴 숫자 낮은 쪽이 승리 |
| 점쟁이 | 🔮 | AI가 낼 카드 미리 공개 |
| 독안개 | ☠️ | 이번 턴 패배 시 골드 2배 손실 |
| 세금 징수 | 💸 | 양측 100G 강제 차감 |
| 황금비 | ✨ | 이번 턴 승자에게 추가 150G 보너스 |
