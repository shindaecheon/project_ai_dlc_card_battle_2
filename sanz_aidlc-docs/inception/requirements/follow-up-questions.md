## Follow-up 1: 게임 규칙 (Question 1.3)

**원래 답변**: "아마 계속 규칙이 바뀔건데, 현재 규칙 기준으로 구현해보자. 나중에 확장될거야."

**명확화 필요**: 현재는 10장 카드, 10 HP, 10턴으로 구현하되, 향후 규칙 변경이 쉽도록 설정 가능하게 만들까요?

A) Yes — 게임 규칙을 상수나 설정 파일로 분리하여 쉽게 변경 가능하게 구현
B) No — 일단 하드코딩으로 빠르게 구현, 나중에 리팩토링
C) 서버 설정으로 관리 — 서버에서 게임 규칙을 설정하고 클라이언트에 전달

[Answer]: B
