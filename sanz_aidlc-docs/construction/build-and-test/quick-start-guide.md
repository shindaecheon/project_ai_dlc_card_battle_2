# Quick Start Guide - 테스트 플레이

## 🚀 빠른 시작 (5분 안에 플레이!)

### 현재 상황
- ✅ 서버 코드 생성 완료
- ✅ 클라이언트 코드 생성 완료
- ⚠️ 구버전 서버 실행 중 (PID: 95603)
- ❌ 새 코드 미반영

---

## Step 1: 서버 재시작 ⚡

### Option A: 터미널에서 직접 재시작 (권장)

서버가 실행 중인 터미널 (pts/4)을 찾아서:

1. **서버 종료**:
   ```
   Ctrl + C
   ```

2. **서버 재시작**:
   ```bash
   cd /home/ec2-user/table-order/server
   npm start
   ```

3. **확인**:
   ```
   ============================================================
   🎮  Card Battle Server
   ============================================================
   ✅ 서버가 시작되었습니다!
   📡 로컬 접속: http://localhost:3000
   🌐 네트워크 접속: http://98.85.250.49:3000
   ```

---

### Option B: 다른 터미널에서 재시작

```bash
# 1. 기존 서버 종료
kill 95603

# 2. 서버 재시작
cd /home/ec2-user/table-order/server
npm start
```

## Step 3: AI 모드 테스트 (1분) 🤖

1. **"🤖 AI 대전"** 버튼 클릭
2. 게임 화면 확인:
   - 플레이어 HP: 10
   - 상대 HP: 10
   - 내 패: 10장 카드
3. **카드 1장 클릭**
4. 배틀 결과 확인:
   - 승리/패배/무승부 메시지
   - HP 변화
5. 계속 플레이 또는 "메인 메뉴로" 클릭

**✅ AI 모드 정상 작동 확인!**

---

## Step 4: 멀티플레이어 테스트 (5분) 🌐

### 준비물
- **브라우저 2개** (탭 2개 또는 다른 기기)

### Player 1 (첫 번째 브라우저)
1. `http://98.85.250.49:3000` 접속
2. **"🌐 멀티플레이어"** 클릭
3. "매칭 중..." 화면 표시 ✅

### Player 2 (두 번째 브라우저/기기)
1. `http://98.85.250.49:3000` 접속
2. **"🌐 멀티플레이어"** 클릭
3. 자동 매칭! 🎉

### 게임 플레이
**양쪽 화면**:
- 게임 시작
- 내 패: 10장 (각자 다름)
- 타이머: 10초
- 이모티콘 버튼 5개

**Player 1**: 카드 1장 클릭
**Player 2**: 카드 1장 클릭

→ 자동으로 배틀 결과 표시! ✅

**추가 테스트**:
- 타이머: 10초 기다리면 자동 제출 ✅
- 이모티콘: 👍 클릭하면 상대방에게 표시 ✅

**✅ 멀티플레이어 정상 작동 확인!**

## 트러블슈팅 🔧

### 문제 1: 모드 선택 화면이 안 보임

**해결책**:
```
브라우저에서 Ctrl + Shift + R
(강력 새로고침으로 캐시 삭제)
```

---

### 문제 2: 서버 재시작이 안 됨

**해결책**:
```bash
# 모든 node 프로세스 종료
pkill -f "node server.js"

# 서버 재시작
cd /home/ec2-user/table-order/server
npm start
```

---

### 문제 3: 매칭이 안 됨

**원인**: 서버 미실행

**해결책**:
```bash
# 서버 실행 확인
ps aux | grep "node server.js"

# 실행 안 되어 있으면
cd /home/ec2-user/table-order/server
npm start
```

---

### 문제 4: "Cannot GET /"

**원인**: 서버가 index.html을 못 찾음

**해결책**:
```bash
# index.html 위치 확인
ls /home/ec2-user/table-order/index.html

# 서버 재시작
cd /home/ec2-user/table-order/server
npm start
```

## 완벽한 첫 게임 플레이 🎮

### 시나리오: 친구와 멀티플레이어

1. **친구에게 URL 공유**:
   ```
   http://98.85.250.49:3000
   ```

2. **함께 접속**:
   - 나: "멀티플레이어" 클릭
   - 친구: "멀티플레이어" 클릭
   - → 자동 매칭! 🎉

3. **게임 플레이**:
   - 카드 선택하며 대화
   - 타이머 보며 긴장감
   - 이모티콘으로 반응 (👍 😊 😢)

4. **게임 종료 후**:
   - 승패 확인
   - "메인 메뉴로" → 다시 플레이!
